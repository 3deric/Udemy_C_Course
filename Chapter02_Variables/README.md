# Chapter 02 Variables

## Variables
-Datatype and Name

-Variable name should be descriptive

-Cant start with a number

-Cant have a space

-Words should be separated with snake_case or CamelCase

-Can be declared and initialized later

## Types

### int
4 Bytes

From: -2*10⁸ To: 2*10⁸

### unsigned int
4Bytes

From: 0, To: 2*10⁹

U after assignment is not needed

### float
4 Bytes

Floating point value

F after assignment is not needed

%f to output a float in a printf statement

### double
8 Bytes

Floating point value

### character
Needs to be declared with single '' not "".

Is internally stored as a numerical value. 'A' = (int)65.



### Operations

Add, Subtract, Divide, Multiply

Division of 2 integers always returns a integer. No matter what the reuslt is assigned to.
At least one float needs to be used in the division to get a float result.
It is better to have both values in a division as float if a float result is required.

### Print Specifier Characters

%i - integer

%u - unsigned integer

%f - float

%lf - double

%x - hexadecimal integer

%c - single character

%s - string of characters
